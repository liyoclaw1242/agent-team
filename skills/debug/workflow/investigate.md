# Investigation Workflow

Reproduce → Trace → Diagnose → Report → Dispatch → Journal

## Phase 1: Reproduce

Read bug report. Set up env. Attempt reproduction.

## Phase 2: Trace

Follow code path. Check git blame. Find related issues.

## Phase 3: Diagnose

Identify root cause. Must pass the "no might be" test.

## Phase 4: Report

Post diagnosis on issue: root cause, affected files, recommended fix, suggested role.

## Phase 5: Dispatch

Create fix bounty for correct role. Close debug issue.
